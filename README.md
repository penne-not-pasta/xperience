# The XPerience.

The xperience is a github repo & website dedicatede to emulating windows XP! This is currently an ongoing project.

## Features

- [x] Boot Screen (*with transition*.)
- [ ] Theme Changer
- [ ] Custom Cursor(s)
- [ ] Windows XP styled windows
- [ ] Working dummy browser with websites.
  - youtube.com
    - may have working videos, may just be clickable thumbnails with links.
  - twitter.com (*fake twitter already logged into a fake account*.)
  - pinterest.com
    - will just embed the actual website
- [ ] working and editable file system
- [ ] games
- [ ] notepad ***OR*** text editor.

> more coming soon.
